1. 터미널(명령 프롬프트)을 엽니다.
2. 프로젝트 폴더로 이동합니다:
   `cd /Users/swg0310bgmail.com/.gemini/antigravity/scratch/femco-shame`
3. Node.js가 설치되어 있는지 확인합니다 (`node -v`).
4. 다음 명령어를 순서대로 실행합니다:

```bash
npm install
npm run dev
```

5. 브라우저에서 `http://localhost:3000` 으로 접속합니다.
